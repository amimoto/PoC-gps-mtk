gps-mtk by amimoto

Perl libraries and scripts for the download and parsing of logged data from MTK chipset based GPS devices.

Dependencies

Only perl and default libraries.

Install

Currently this project is in development so aside from pulling the code directly from git, there's no convenient way of doing so. It's ready for beta testing, it'll become available as a tarball.
